Як ми працюємо

Дізнайтесь більше про механізм нашої співпраці

Крок 1: Замовлення поїздки / Надання контактів для зв'язку

Клієнт робить замовлення поїздки або надає свої контактні дані для подальшого зв'язку. Інформація, яку надає клієнт: адреса відправлення, адреса призначення, дата та час поїздки, особливі вимоги або інша необхідна інформація.

Крок 2: Зв'язок з менеджером для уточнення деталей поїздки. Надання необхідної інформації

Менеджер зв'язується з клієнтом для уточнення деталей поїздки. Можливі деталі для уточнення: точна адреса відправлення, точна адреса призначення, додаткові вимоги або послуги, які можуть знадобитися, кількість пасажирів тощо. Клієнт надає менеджеру необхідну інформацію для оптимальної організації поїздки.

Крок 3: Розрахунок вартості поїздки

На основі наданої інформації менеджер розраховує вартість поїздки і повідомляє клієнта. Клієнт може внести оплату або узгодити деталі оплати.

Крок 4: Поїздка

Заберемо за адресою відправлення, доставимо до пункту призначення: По зазначеному часу транспорт забирає клієнта за адресою відправлення. Здійснюється поїздка до пункту призначення відповідно до узгодженої інформації. По завершенні поїздки клієнт може залишити відгук та оплатити послугу, якщо цього не було зроблено раніше.
